# e-commerce


>## Task description

### Create a web platform that will be able to:

* Add / remove good to cart
* Checkout
* Manage the platform from the admin panel (Add, update, delete goods, view orders)

>## Project setup

#### 1. cd be
#### 2. npm i
#### 3. cd fe
#### 4. npm i


>## To run the project:

### Back-end

* #### `cd be`
* #### `npm run start`

### Front-end

* #### `cd fe`
* #### `npm run start`
